# Product Overview

You are building a web application called:

"Tour Reminder & Customer Communication System"

## Purpose
The system helps travel operators manage bookings and send automated reminders to customers about upcoming tours.

## Core Features
- Booking management (imported via CSV or email parsing)
- Automated reminders (H-1 day, H-3 hours)
- Customer communication (chat or WhatsApp integration)
- Admin dashboard

## Target Users
- Global tourists (non-technical users)
- Travel operators (admin)

## UX Goal
- Reduce anxiety before tours
- Make information clear and accessible
- Provide quick communication access

## Key UX Priorities
- Show time clearly
- Show meeting point clearly
- Provide instant contact access
