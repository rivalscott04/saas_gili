# Design Principles

## 1. Clarity First
Always prioritize readability and clarity over visual complexity.

## 2. Minimal Cognitive Load
Users should understand the interface within 3 seconds.

## 3. Consistency
All components must follow the same spacing, color, and behavior rules.

## 4. Visual Hierarchy
Important information must stand out:
- Tour time
- Countdown
- Action buttons

## 5. Travel-Friendly Feel
Design should feel:
- Calm
- Trustworthy
- Slightly adventurous

## 6. Accessibility
- High contrast text
- Minimum font size: 14px
- Buttons easily clickable

## 7. No Overdesign
Avoid:
- excessive gradients
- too many colors
- complex layouts
